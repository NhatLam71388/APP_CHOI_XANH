## 🛠 Công nghệ sử dụng
Dự án ứng dụng mô hình phát triển đa nền tảng (Hybrid App) giúp tối ưu thời gian triển khai trên cả Android và iOS.

| Thành phần | Công nghệ & Kỹ thuật |
| :--- | :--- |
| **Mobile Framework** | **Flutter** (Dart) |
| **Backend** | **.NET Core** & **RESTful API** |
| **Authentication** | **JWT (JSON Web Token)** |
| **Rendering** | Kết hợp **CSR** (Client-Side) & **SSR** (Server-Side Rendering) |
| **Optimization** | Cơ chế **Cache** giúp tăng tốc độ tải trang và giảm tải Server |
| **Content Tools** | Tích hợp **CKEditor** & **CKFinder** quản lý nội dung |
