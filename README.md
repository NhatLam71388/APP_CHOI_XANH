# 📱 Ứng Dụng Mua Bán Trực Tuyến - E-Commerce Mobile App

---

# 🛠 Công nghệ sử dụng

Dự án ứng dụng mô hình phát triển đa nền tảng (Hybrid App) giúp tối ưu thời gian triển khai trên cả Android và iOS.

| Thành phần | Công nghệ & Kỹ thuật |
| :--- | :--- |
| **Mobile Framework** | **Flutter (Dart)** |
| **Backend** | **.NET Core & RESTful API** |
| **Authentication** | **JWT (JSON Web Token)** |
| **Rendering** | **CSR & SSR** |
| **Optimization** | **Cache** |
| **Content Tools** | **CKEditor & CKFinder** |
