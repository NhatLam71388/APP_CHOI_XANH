## 📱 Demo Ứng Dụng

### Mục tiêu: Xây dựng trang chủ và tích hợp dữ liệu thực tế
[cite_start]Trong giai đoạn này, dự án tập trung vào việc hình thành khung xương cho ứng dụng thương mại điện tử và đảm bảo luồng dữ liệu thông suốt từ server[cite: 278, 279].

#### Các công việc chính:
* [cite_start]**Thiết kế UI/UX:** Sử dụng framework **Flutter** để xây dựng giao diện trang chủ đa nền tảng, đảm bảo tính thẩm mỹ và hiệu suất cao[cite: 281].
* [cite_start]**Tích hợp API động:** Kết nối thành công với hệ thống API nội bộ để lấy danh sách sản phẩm thực tế bao gồm: hình ảnh, tên sản phẩm và giá bán[cite: 279, 282].
* [cite_start]**Tối ưu hóa Model:** Xử lý kỹ thuật tinh lọc dữ liệu (mapping) từ API, loại bỏ các trường thừa để tối ưu bộ nhớ và giúp ứng dụng vận hành mượt mà hơn[cite: 283, 284].
